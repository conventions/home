JSF Conventions Framework ShowCase

JSF Conventions Framework is a set of components based on top of CDI/CODI, EJB, JSF, Primefaces and Hibernate build to facilitate the development of rich web applications.
  • Fast and easy crud creation.
  • True pagination out of the box.
  • Built in security (method authorization, page access and page block restrictions).
  • Built in stateless, stateful and non EJB Crud solution.
  • Code generation.
  • Non JavaEE application server support.
  • Set of UI components based on Primefaces.
  • Internationalization.
  • Exception Handling.
  • ...and much more
